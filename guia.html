Tareas comunes

Crear una encuesta en Qualtrics

Sigue los pasos para crear una encuesta básica:

  1. Haz clic en "Create Project".
  2. Selecciona "Survey" y asígnale un nombre.
  3. Agrega tus preguntas usando el editor visual.

Código para lógica condicional:

if (${q://QID1/ChoiceTextEntryValue} == "Sí") {
  Qualtrics.SurveyEngine.setEmbeddedData("continuar", true);
}

Usar condiciones lógicas

Aprende a mostrar u ocultar preguntas según las respuestas del usuario:

  1. Selecciona la pregunta que quieres condicionar.
  2. Haz clic en "Add Display Logic".
  3. Configura la condición según la respuesta de otra pregunta.

Ejemplo de lógica condicional:

if (${q://QID2/ChoiceTextEntryValue} == "No") {
  Qualtrics.SurveyEngine.setEmbeddedData("mostrar_pregunta", false);
}

Añadir loops

Utiliza la función de Loop & Merge para repetir bloques de preguntas:

  1. Haz clic en el bloque donde quieres aplicar el loop.
  2. Selecciona "Loop & Merge" en las opciones del bloque.
  3. Configura los valores o elementos para iterar.
